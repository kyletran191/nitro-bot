### Installation
- Install
  ```
  download python: https://www.python.org/downloads/
  pip install -r requirements.txt
  ```

- Clone this repository -
  ```
  git clone https://github.com/kyletran191/nitro-bot.git
  ```
- Now go to run `bot.py` -
  ```
  $ cd nitro-bot
  $ python bot.py
  ```
